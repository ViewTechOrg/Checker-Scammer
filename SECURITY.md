# Security Policy

## Supported Versions

We strive to maintain the security of this project and provide security updates for the latest version available in this repository.

| Version        | Supported |
| -------------- | --------- |
| Latest         | ✅         |
| Older versions | ❌         |

---

## Reporting a Vulnerability

If you discover a security vulnerability, sensitive bug, or potential abuse in this project, please **do not open a public issue** to avoid exposing vulnerability details.

Please report it privately via Telegram:

**Telegram:** @ScriptingCli21

Please include the following information to help speed up the investigation:

* Short description of the vulnerability
* Steps to reproduce (if possible)
* Possible impact
* Supporting evidence (screenshots, logs, payloads, etc.)
* Mitigation suggestions (optional)

---

## Response Time

We will try to respond as quickly as possible after receiving the report. Response time may vary depending on the complexity of the issue.

---

## Responsible Disclosure

We highly appreciate responsible disclosure. Please allow time for verification and fixes before publicly disclosing vulnerability details.

---

Thank you for helping keep this project secure.

# Kebijakan Keamanan

## Versi yang Didukung

Kami berusaha menjaga keamanan project ini dan akan memberikan pembaruan keamanan untuk versi terbaru yang tersedia di repository ini.

| Versi      | Didukung |
| ---------- | -------- |
| Terbaru    | ✅        |
| Versi lama | ❌        |

---

## Melaporkan Kerentanan

Jika Anda menemukan celah keamanan, bug sensitif, atau potensi penyalahgunaan pada project ini, mohon **jangan membuka issue publik** agar detail kerentanan tidak terekspos.

Silakan laporkan secara pribadi melalui Telegram:

**Telegram:** @ScriptingCli21

Mohon sertakan informasi berikut agar proses investigasi lebih cepat:

* Deskripsi singkat kerentanan
* Langkah reproduksi (jika memungkinkan)
* Dampak yang mungkin terjadi
* Bukti pendukung (screenshot, log, payload, dll.)
* Saran mitigasi (opsional)

---

## Waktu Respon

Kami akan berusaha memberikan respon secepat mungkin setelah laporan diterima. Waktu respon dapat bervariasi tergantung tingkat kompleksitas masalah.

---

## Pengungkapan yang Bertanggung Jawab

Kami sangat menghargai responsible disclosure. Mohon beri waktu untuk verifikasi dan perbaikan sebelum mempublikasikan detail kerentanan kepada publik.

---

Terima kasih telah membantu menjaga keamanan project ini.
