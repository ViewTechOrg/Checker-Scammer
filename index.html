Checker Scammer

Kenali dengan baik sebelum anda tertipu oleh hal yang tampak meyakinkan.

Dokumentasi

Alat penyelidikan berbasis CLI untuk mengidentifikasi nomor WhatsApp yang berpotensi scammer

Apa itu Checker Scammer?

Checker-Scammer adalah alat Penyelidikan berbasis CLI yang memanfaatkan teknik Perhitungan berdasarkan jumlah tag dalam batas tertentu, berfungsi untuk mengidentifikasi sebuah nomor platform whatsapp apakah masuk golongan scammer atau seseorang yang aman sebelum memulai transaksi, di era sekarang yang serba online tidak sedikit pula aksi kejahatan seperti scammer dan penipuan banyak bermunculan, dengan adanya alat ini di harapkan dapat membantu mengatasi masalah penipuan/scam saat bertransaksi.

Fitur Utama

Analisis Nomor Telepon

Analisis nomor telepon secara cepat dan ringan melalui CLI untuk mendeteksi indikasi penipuan.

Data yang Dikumpulkan

Checker-Scammer Menampilkan foto profil, informasi umum seperti nomor HP, serta hasil analisis berdasarkan algoritma untuk mengidentifikasi potensi sebagai victim atau pihak yang pernah terlibat dalam aktivitas penipuan.

Cara Install & Jalankan

1. Instalasi

Buka terminal dan jalankan perintah berikut sesuai dokumentasi:

pkg update
pkg upgrade
apt install git make just
git clone https://github.com/ViewTechOrg/Checker-Scammer
cd Checker-Scammer
make install
just run

2. Pengaturan Layar

Portrait Mode

x: 101, y: 35

Landscape Mode

x: 181, y: 70

Untuk mengatur ukuran, cukup mencubit layar (zoom gesture) jika menggunakan perangkat sentuh.

3. Kontrol

  • Gunakan kursor atau tombol panah untuk mengontrol alat
  • Gunakan keyboard untuk mengetik di kolom input
  • Gunakan tools ini saat ingin memverifikasi nomor seseorang sebelum melakukan transaksi
  • Selamat! Anda bisa bertransaksi dengan lebih aman. (Sedia Payung Sebelum Hujan)
